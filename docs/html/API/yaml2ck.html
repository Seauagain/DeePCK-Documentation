yaml2ck¶

Convert .yaml format to CHEMKIN format.

yaml2ck.main()¶

Convert .yaml to .inp, trans.dat and thermo.dat.

Examples

>>> python yaml2ck.py --input='gri.yaml' --ouput='gri.inp'
Convert 'gri.yaml' to 'gri.inp', 'gri_thermo.dat' and 'gri_transport.dat'.
